# Utlc and Stlc in Typescript

This is an implementation of [Checking Dependent Types with Normalization by Evaluation: A Tutorial](https://davidchristiansen.dk/tutorials/nbe/) in TypeScript.

## Build
```shell
npm install
npm run build
```

## Run Unit Tests
```shell
npm test
```
